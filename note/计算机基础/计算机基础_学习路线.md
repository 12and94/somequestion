# 计算机基础：学习路线（分步实现）

本文对应 `面经题目分类汇总.md` 的 **三、计算机基础（3.1~3.5）**。

## 文档策略
- 每个专题一个独立 `md` 文件，避免内容过载。
- 每题统一结构：
- 标准准确的说法
- 通俗易懂的理解
- 面试回答简版模板
- 对应示例（本阶段可先以 note 为主）

## 分批顺序（高频优先）
1. 第一批：`3.1 内存与进程`、`3.2 多线程与同步`
2. 第二批：`3.3 网络`
3. 第三批：`3.4 编译与链接`、`3.5 其他系统知识`

## 加厚规则
- 先讲“定义/原理”，再讲“工程影响”。
- 面试高频题必须补“误区与边界”（例如：`volatile` 不是线程同步）。
- 复杂系统题优先用流程分解回答：概念 -> 机制 -> 代价 -> 取舍。
