# 物理引擎 6：动画联动与特殊模拟

## 1. 动画与物理的关系
### 1.1 动画驱动物理
- 动画先行，物理做补偿（常见角色主控制）。

### 1.2 物理驱动动画
- 物理结果驱动骨骼（ragdoll、布料附着）。

## 2. 角色控制器（Character Controller）
- 常见策略：胶囊体 + Sweep + 地面探测。
- 关键问题：台阶、坡面、滑墙、挤压、边缘稳定。
- 为什么不用纯动态刚体做主角：可控性、可预测性、联机一致性更难。

## 3. Ragdoll
- 骨骼关节约束集合。
- 难点：和动画状态切换平滑、穿模修正、网络同步开销。

## 4. 布料/绳子/软体
- 实时常见 PBD/XPBD：
  - 优点：稳定、易控制。
  - 缺点：严格物理精度有限。
- 常见工程技巧：分级模拟、距离 LOD、碰撞简化。

## 5. 与渲染协作
- 法线、切线、蒙皮更新时序要与物理写回一致。
- 透明特效与物理粒子往往是性能敏感区。
