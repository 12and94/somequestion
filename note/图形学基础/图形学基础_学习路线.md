# 图形学基础：学习路线（分步实现）

本文对应 `面经题目分类汇总.md` 的 **二、图形学基础（2.1~2.13）**。

## 文档策略
- 每个专题一个独立 `md` 文件，避免大杂烩。
- 每题统一结构：
- 标准准确的说法
- 通俗易懂的理解
- 面试回答简版模板
- 难点深挖（流程图/关键公式/工程坑点）
- 图形学专题本阶段可以不写 `case`，重点把“答题质量”先打实。

## 分批顺序（高频优先）
1. 第一批：`2.1 渲染管线`、`2.2 深度测试与Early-Z`、`2.3 空间变换与坐标系`
2. 第二批：`2.4 光照模型与PBR`、`2.5 阴影技术`、`2.6 纹理与Mipmap`
3. 第三批：`2.7 全局光照`、`2.8 抗锯齿`、`2.9 剔除与优化`
4. 第四批：`2.10 Vulkan/OpenGL/DX`、`2.11 动画与蒙皮`
5. 第五批：`2.12 数学基础`、`2.13 其他图形学`

## 难点加厚规则
- 高频且容易追问的题，增加“追问链路”：原理 -> 性能代价 -> 工程取舍。
- 复杂流程题（渲染管线、Early-Z、MVP、阴影、TAA、Vulkan同步）优先加 Mermaid 图。
- 容易答虚的题（如 BRDF、RenderPass、IBL）强制给“最小闭环”表述，避免口号式答案。
