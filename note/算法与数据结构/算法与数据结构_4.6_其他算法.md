## 题目：两个栈实现队列

### 标准准确的说法
- 用 `in` 栈负责入队，`out` 栈负责出队。
- 出队/队头操作时若 `out` 为空，把 `in` 全部倒入 `out`。
- 均摊复杂度：`push O(1)`，`pop/peek` 均摊 `O(1)`。

### 通俗易懂的理解
- 一个栈专门“收货”，另一个栈专门“发货”；倒一次货后可以连续出队很多次。

### 面试回答简版模板
`两个栈实现队列的关键是延迟倒栈：只在 out 为空时才把 in 全量搬运，因此操作均摊 O(1)。`

### 对应示例
```cpp
class MyQueue {
    std::stack<int> in_, out_;
    void move() {
        if (out_.empty()) {
            while (!in_.empty()) {
                out_.push(in_.top());
                in_.pop();
            }
        }
    }
public:
    void push(int x) { in_.push(x); }
    int pop() { move(); int v = out_.top(); out_.pop(); return v; }
    int peek() { move(); return out_.top(); }
    bool empty() const { return in_.empty() && out_.empty(); }
};
```

## 题目：判断点在三角形内（算法实现）

### 标准准确的说法
- 常见解法：叉积同向法。
- 对三角形边 AB、BC、CA，检查点 P 是否都在同一侧。
- 允许边界时可用 `>=0`/`<=0` 容差判断。
- 注意浮点误差，工程中常加 epsilon。

### 通俗易懂的理解
- 看点 P 相对三条边是不是都“站在同一边”，是就说明在三角形里。

### 面试回答简版模板
`点在三角形内可用叉积同向判定：分别看 P 对三条边的方向符号是否一致，并用 epsilon 处理浮点误差。`

### 对应示例
```cpp
struct Vec2 { double x, y; };

double cross(const Vec2& a, const Vec2& b, const Vec2& c) {
    // (b-a) x (c-a)
    return (b.x - a.x) * (c.y - a.y) - (b.y - a.y) * (c.x - a.x);
}

bool inTriangle(const Vec2& A, const Vec2& B, const Vec2& C, const Vec2& P) {
    const double eps = 1e-9;
    double c1 = cross(A, B, P), c2 = cross(B, C, P), c3 = cross(C, A, P);
    bool nonNeg = (c1 >= -eps && c2 >= -eps && c3 >= -eps);
    bool nonPos = (c1 <= eps && c2 <= eps && c3 <= eps);
    return nonNeg || nonPos;
}
```

## 题目：零钱兑换（动态规划）

### 标准准确的说法
- 经典完全背包：`dp[x]` 表示凑出金额 x 的最少硬币数。
- 转移：`dp[x] = min(dp[x], dp[x-coin] + 1)`。
- 初始化：`dp[0]=0`，其他为 INF。
- 复杂度：`O(amount * coins.size())`。

### 通俗易懂的理解
- 从 0 元开始递推，每多 1 元都问“最后一枚硬币选谁最划算”。

### 面试回答简版模板
`零钱兑换最少硬币数用一维 DP：dp[x] 由 dp[x-coin]+1 转移，属于完全背包模型。`

### 对应示例
```cpp
int coinChange(std::vector<int>& coins, int amount) {
    const int INF = 1e9;
    std::vector<int> dp(amount + 1, INF);
    dp[0] = 0;
    for (int x = 1; x <= amount; ++x) {
        for (int c : coins) {
            if (x >= c && dp[x - c] != INF) {
                dp[x] = std::min(dp[x], dp[x - c] + 1);
            }
        }
    }
    return dp[amount] == INF ? -1 : dp[amount];
}
```

## 题目：随机洗牌算法

### 标准准确的说法
- Fisher-Yates 洗牌：从后往前，`i` 与 `[0..i]` 随机位置交换。
- 该算法能保证每种排列等概率出现（在均匀随机数前提下）。
- 复杂度：`O(N)`。

### 通俗易懂的理解
- 每一步从“还没固定的位置”里随机抽一个放到当前位置。

### 面试回答简版模板
`随机洗牌应使用 Fisher-Yates，从后向前随机交换 i 与 [0..i]，可保证等概率。`

### 对应示例
```cpp
void shuffle(std::vector<int>& a) {
    std::mt19937 rng(std::random_device{}());
    for (int i = (int)a.size() - 1; i > 0; --i) {
        std::uniform_int_distribution<int> dist(0, i);
        int j = dist(rng);
        std::swap(a[i], a[j]);
    }
}
```

## 题目：随机红包算法

### 标准准确的说法
- 典型目标：总额固定、个数固定、每份随机且不低于最小值。
- 常见实现：
- “二倍均值法”或“区间随机 + 剩余兜底”。
- 关键约束：
- 后续每人至少保留最小金额（避免前面发太多导致后面不合法）。

### 通俗易懂的理解
- 发红包不能只图随机，还要保证“后面每个人还有保底”。

### 面试回答简版模板
`随机红包要在随机性和约束合法性间平衡，核心是每次分配都预留后续最小保底，最后一人拿剩余。`

### 对应示例
```cpp
std::vector<int> randomRedPacket(int total, int n, int minVal = 1) {
    std::vector<int> ans;
    std::mt19937 rng(std::random_device{}());
    int remain = total;
    for (int i = 1; i < n; ++i) {
        int maxGive = remain - (n - i) * minVal;
        int curMax = std::max(minVal, std::min(maxGive, 2 * remain / (n - i + 1)));
        std::uniform_int_distribution<int> dist(minVal, curMax);
        int x = dist(rng);
        ans.push_back(x);
        remain -= x;
    }
    ans.push_back(remain);
    return ans;
}
```

## 题目：蒙特卡洛求圆周率

### 标准准确的说法
- 在正方形内随机撒点，统计落在内切圆内的比例。
- 比例约为 `πr^2 / (2r)^2 = π/4`，故 `π ≈ 4 * inside / total`。
- 误差随样本量增大而减小，收敛较慢（随机方法特性）。

### 通俗易懂的理解
- 随机扔飞镖，落在圆里的比例越接近真实面积比，π 估计越准。

### 面试回答简版模板
`蒙特卡洛估计 π 用面积比思想：在单位正方形随机采样，圆内点占比乘 4 得到 π 近似。`

### 对应示例
```cpp
double estimatePiMonteCarlo(int samples) {
    std::mt19937 rng(std::random_device{}());
    std::uniform_real_distribution<double> dist(0.0, 1.0);
    int inside = 0;
    for (int i = 0; i < samples; ++i) {
        double x = dist(rng), y = dist(rng);
        if (x * x + y * y <= 1.0) ++inside;
    }
    return 4.0 * inside / samples;
}
```

## 题目：牛顿法求平方根

### 标准准确的说法
- 求 `sqrt(S)` 可转成解方程 `f(x)=x^2-S=0`。
- 牛顿迭代：`x_{n+1} = (x_n + S/x_n) / 2`。
- 收敛快（二次收敛），但要设终止条件和初值。

### 通俗易懂的理解
- 每次用“当前猜测”和“反推值 S/x”取平均，猜测会很快逼近真值。

### 面试回答简版模板
`牛顿法把开方转为求根迭代，公式 x=(x+S/x)/2，收敛快，需注意精度阈值与 0/负数边界。`

### 对应示例
```cpp
double sqrtNewton(double s) {
    if (s < 0) return std::numeric_limits<double>::quiet_NaN();
    if (s == 0) return 0;
    double x = s;
    for (int i = 0; i < 100; ++i) {
        double nx = 0.5 * (x + s / x);
        if (std::abs(nx - x) < 1e-12) return nx;
        x = nx;
    }
    return x;
}
```

## 题目：约瑟夫环

### 标准准确的说法
- 经典递推：`f(1)=0`，`f(n)=(f(n-1)+k)%n`（0-based）。
- 最终答案若题目 1-based 需 `+1`。
- 复杂度：`O(N)` 时间，`O(1)` 空间。

### 通俗易懂的理解
- 每轮淘汰后，相当于“起点平移”，递推公式就是在累计这个平移量。

### 面试回答简版模板
`约瑟夫环最优常用递推 f(n)=(f(n-1)+k)%n，线性时间常数空间可得最后幸存者。`

### 对应示例
```cpp
int josephus(int n, int k) { // 返回 1-based
    int f = 0;
    for (int i = 2; i <= n; ++i) f = (f + k) % i;
    return f + 1;
}
```

## 题目：数据流中位数

### 标准准确的说法
- 双堆法：
- 大顶堆 `left` 存较小一半。
- 小顶堆 `right` 存较大一半。
- 维护平衡：`left.size()` 与 `right.size()` 差不超过 1，且 `left.top() <= right.top()`。
- 复杂度：插入 `O(log N)`，查询中位数 `O(1)`。

### 通俗易懂的理解
- 用两堆把数据分成“左半”和“右半”，中位数就在两堆堆顶附近。

### 面试回答简版模板
`数据流中位数用双堆维护左右半区：插入时重平衡，查询时根据堆大小取堆顶或均值。`

### 对应示例
```cpp
class MedianFinder {
    std::priority_queue<int> left; // max-heap
    std::priority_queue<int, std::vector<int>, std::greater<int>> right; // min-heap
public:
    void addNum(int x) {
        if (left.empty() || x <= left.top()) left.push(x);
        else right.push(x);

        if (left.size() > right.size() + 1) {
            right.push(left.top()); left.pop();
        } else if (right.size() > left.size()) {
            left.push(right.top()); right.pop();
        }
    }

    double findMedian() const {
        if (left.size() == right.size()) return (left.top() + right.top()) / 2.0;
        return left.top();
    }
};
```

## 题目：24点游戏

### 标准准确的说法
- 常用回溯：每次从当前数字集合里选两个数做四则运算，生成新集合递归。
- 终止条件：只剩一个数且接近 24（浮点误差容差）。
- 关键：
- 处理除零。
- 去重（交换律导致重复状态，可做排序或状态剪枝）。

### 通俗易懂的理解
- 从 4 个数字开始，不断“任取两数算一次再塞回去”，直到只剩一个结果看是不是 24。

### 面试回答简版模板
`24 点常用 DFS 回溯：两两选数做 + - * /，递归尝试所有可能，最后用 epsilon 判断是否到 24，并注意除零和去重。`

### 对应示例
```cpp
bool dfs24(std::vector<double> nums) {
    const double EPS = 1e-6;
    if (nums.size() == 1) return std::abs(nums[0] - 24.0) < EPS;
    int n = (int)nums.size();
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            if (i == j) continue;
            std::vector<double> nxt;
            for (int k = 0; k < n; ++k) if (k != i && k != j) nxt.push_back(nums[k]);

            // 枚举 i op j
            std::vector<double> cand{nums[i] + nums[j], nums[i] - nums[j], nums[i] * nums[j]};
            if (std::abs(nums[j]) > EPS) cand.push_back(nums[i] / nums[j]);

            for (double v : cand) {
                nxt.push_back(v);
                if (dfs24(nxt)) return true;
                nxt.pop_back();
            }
        }
    }
    return false;
}
```
