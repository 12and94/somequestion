# 算法与数据结构学习路线

## 第一轮：高频必会（先把模板背熟）
1. 链表：反转链表、删除倒数第 N 个、环检测
2. 数组与哈希：两数之和、三数之和、接雨水、LRU
3. 排序与查找：快排、二分、TopK、堆排序
4. 树：遍历、平衡树判断、前中后序互推

## 第二轮：进阶与易错
1. 字符串：KMP、括号匹配、AC 自动机
2. 其他算法：零钱兑换、数据流中位数、约瑟夫环、随机洗牌
3. 设计题：对象池、排行榜、定时器、线程池

## 第三轮：面试强化
1. 每题至少能说：思路、复杂度、边界情况、代码模板
2. 对比题重点：红黑树 vs AVL、快排 vs 归并、TCP/UDP 场景化选型
3. 对设计题补充：并发安全、可扩展、监控指标
