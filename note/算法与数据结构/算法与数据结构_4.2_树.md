## 题目：前序+中序求后序遍历

### 标准准确的说法
- 前序第一个元素是根。
- 在中序里找到根位置，可分出左子树和右子树区间。
- 递归处理左右子树，最后输出根（后序）。
- 复杂度：
- 朴素查找根位置 `O(N^2)`。
- 预先用哈希表存中序下标可降为 `O(N)`。

### 通俗易懂的理解
- 前序告诉你“谁先当老板”，中序告诉你“左团队和右团队怎么分”，递归分完后按“左-右-根”吐出来就是后序。

### 面试回答简版模板
`前序首元素定根，中序定位根切左右区间，递归构造后序。用 index map 可把时间复杂度优化到 O(N)。`

### 对应示例
```cpp
void buildPost(const std::vector<int>& pre, int pl, int pr,
               const std::vector<int>& in, int il, int ir,
               const std::unordered_map<int, int>& pos,
               std::vector<int>& post) {
    if (pl > pr) return;
    int root = pre[pl];
    int k = pos.at(root);
    int leftSize = k - il;
    buildPost(pre, pl + 1, pl + leftSize, in, il, k - 1, pos, post);
    buildPost(pre, pl + leftSize + 1, pr, in, k + 1, ir, pos, post);
    post.push_back(root);
}
```

## 题目：判断平衡二叉树

### 标准准确的说法
- 平衡二叉树定义：任意节点左右子树高度差不超过 1。
- 后序 DFS 同时返回“子树高度 + 是否平衡”。
- 优化写法：若某子树已不平衡直接返回 -1，向上快速剪枝。
- 复杂度：`O(N)` 时间，`O(H)` 递归栈空间。

### 通俗易懂的理解
- 像检查每个节点是不是“左右肩膀高差 <= 1”，只要某处失衡整棵树就不平衡。

### 面试回答简版模板
`用后序遍历自底向上算高度并校验平衡，失衡返回 -1 剪枝，一次 DFS 即可完成，O(N)。`

### 对应示例
```cpp
struct TreeNode { int val; TreeNode* left; TreeNode* right; };

int heightOrNeg(TreeNode* root) {
    if (!root) return 0;
    int lh = heightOrNeg(root->left);
    if (lh == -1) return -1;
    int rh = heightOrNeg(root->right);
    if (rh == -1) return -1;
    if (std::abs(lh - rh) > 1) return -1;
    return std::max(lh, rh) + 1;
}

bool isBalanced(TreeNode* root) { return heightOrNeg(root) != -1; }
```

## 题目：二叉树的前序/中序/后序遍历

### 标准准确的说法
- 三种 DFS 顺序区别：
- 前序：根-左-右。
- 中序：左-根-右。
- 后序：左-右-根。
- 递归写法简洁，迭代写法更考察栈模拟能力。

### 通俗易懂的理解
- 把“访问根节点”放在不同位置，就得到前中后三种遍历。

### 面试回答简版模板
`前中后序本质是 DFS 中“访问根”的时机不同。面试可先给递归，再补一版栈迭代。`

### 对应示例
```cpp
void preorder(TreeNode* r, std::vector<int>& ans) {
    if (!r) return;
    ans.push_back(r->val);
    preorder(r->left, ans);
    preorder(r->right, ans);
}

void inorder(TreeNode* r, std::vector<int>& ans) {
    if (!r) return;
    inorder(r->left, ans);
    ans.push_back(r->val);
    inorder(r->right, ans);
}

void postorder(TreeNode* r, std::vector<int>& ans) {
    if (!r) return;
    postorder(r->left, ans);
    postorder(r->right, ans);
    ans.push_back(r->val);
}
```

## 题目：BFS和DFS遍历二叉树

### 标准准确的说法
- BFS：按层遍历，用队列。
- DFS：沿分支深入，用递归或栈。
- 场景差异：
- BFS 适合最短步数、层级问题。
- DFS 适合路径、回溯、子树性质统计。

### 通俗易懂的理解
- BFS 像“同层一起扫楼”，DFS 像“先把一条路走到底再回头”。

### 面试回答简版模板
`BFS 用队列做层序，DFS 用递归/栈做深度遍历。选择取决于题目目标：层信息常选 BFS，路径与回溯常选 DFS。`

### 对应示例
```cpp
std::vector<std::vector<int>> levelOrder(TreeNode* root) {
    std::vector<std::vector<int>> ans;
    if (!root) return ans;
    std::queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int sz = (int)q.size();
        ans.push_back({});
        while (sz--) {
            TreeNode* cur = q.front(); q.pop();
            ans.back().push_back(cur->val);
            if (cur->left) q.push(cur->left);
            if (cur->right) q.push(cur->right);
        }
    }
    return ans;
}
```

```cpp
void dfs(TreeNode* root, std::vector<int>& order) {
    if (!root) return;
    order.push_back(root->val);
    dfs(root->left, order);
    dfs(root->right, order);
}
```

## 题目：红黑树和AVL树的区别

### 标准准确的说法
- AVL：高度平衡更严格（任意节点平衡因子 -1/0/1）。
- 红黑树：平衡约束较宽松，保证最长路径不超过最短路径的 2 倍量级。
- 复杂度：查找/插入/删除都为 `O(log N)`。
- 工程差异：
- AVL 查询更稳，但插删旋转可能更多。
- 红黑树插删调整更温和，工程库中更常见（如很多 map/set 实现）。

### 通俗易懂的理解
- AVL 像“严格管理队形”，红黑树像“允许小波动但整体不乱”。

### 面试回答简版模板
`AVL 平衡更严格，查询高度更优但更新代价偏高；红黑树平衡约束更松，插删调整更少，工程容器实现中更常见。`

### 对应示例
```cpp
// AVL 旋转示意（右旋）
Node* rightRotate(Node* y) {
    Node* x = y->left;
    Node* T2 = x->right;
    x->right = y;
    y->left = T2;
    // 更新高度 ...
    return x;
}
```

## 题目：堆的结构、插入和删除

### 标准准确的说法
- 堆是完全二叉树，常用数组表示。
- 最大堆性质：父节点 >= 子节点；最小堆反之。
- 插入：新元素放末尾，上滤（sift up）。
- 删除堆顶：末尾元素换到堆顶，下滤（sift down）。
- 复杂度：插入/删除 `O(log N)`，取堆顶 `O(1)`，建堆 `O(N)`。

### 通俗易懂的理解
- 堆像“冠军台”：堆顶是当前冠军，插入靠比拼上位，删除冠军后由替补下沉调整。

### 面试回答简版模板
`堆是数组实现的完全二叉树，插入上滤、删顶下滤，单次调整 O(log N)，批量建堆 O(N)。`

### 对应示例
```cpp
class MaxHeap {
public:
    void push(int x) {
        a.push_back(x);
        siftUp((int)a.size() - 1);
    }

    int top() const { return a.front(); }

    void pop() {
        if (a.empty()) return;
        std::swap(a[0], a.back());
        a.pop_back();
        if (!a.empty()) siftDown(0);
    }

private:
    std::vector<int> a;

    void siftUp(int i) {
        while (i > 0) {
            int p = (i - 1) / 2;
            if (a[p] >= a[i]) break;
            std::swap(a[p], a[i]);
            i = p;
        }
    }

    void siftDown(int i) {
        int n = (int)a.size();
        while (true) {
            int l = 2 * i + 1, r = 2 * i + 2, mx = i;
            if (l < n && a[l] > a[mx]) mx = l;
            if (r < n && a[r] > a[mx]) mx = r;
            if (mx == i) break;
            std::swap(a[i], a[mx]);
            i = mx;
        }
    }
};
```
